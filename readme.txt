Основная идея этого проекта - получить информацию из загруженного excel файла и вбить ее в поля на сайте "https://rpachallenge.com/".

Вероятные проблемы - решения:
Файлы старые, другого формата - дополнительно форматер сделать или попробовать найти другую библиотеку.
Вероятно у пользователя установлен не Google Chrome - Сбилдить проект в докере. Или в init SeleniumController принимать строкой требуемый браузер и выбирать нужный вебдрайвер.
Больше одного пользователя работает - при загрузке проверять что файл с таким именем еще не существует, если есть - добавить "(1)" к названию.
